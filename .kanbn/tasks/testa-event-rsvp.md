# testa-event-rsvp

1. Navigera till Events-fliken
2. Bekräfta att evenemang laddas från backend
3. Öppna ett event — granska detaljer (datum, plats, beskrivning)
4. Skicka in en RSVP
5. Gå till "Mina RSVPs" — verifiera att anmälan syns
6. Uppdatera eller avboka RSVP

**Förväntat:** Events hämtas, RSVP-flödet fungerar, ändringar reflekteras direkt
