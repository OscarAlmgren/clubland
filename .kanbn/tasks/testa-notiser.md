# Testa notiser

1. Trigga en händelse som genererar en notis (t.ex. bokning bekräftad)
2. Öppna notis-panelen
3. Bekräfta att notisen visas med rätt text och tidsstämpel
4. Markera notisen som läst — badge-räknaren uppdateras
5. Markera alla som lästa

**Förväntat:** Notiser laddas från backend, läst/oläst-status synkas korrekt
